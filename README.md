# realtime-weather
realtime weather view using Socket.io



### Clone and Run Locally
```
$ git clone https://github.com/KBPsystem777/realtime-weather

$ cd realtime-weather

$ npm install

$ npm run build

$ npm run start
```
App will be available on http://localhost:7000

### Dependencies
```
"axios": "^0.18.0"
"browserify": "^16.2.2"
"express": "^4.16.3"
"socket.io": "^2.1.1"
```
