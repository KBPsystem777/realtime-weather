# realtime-weather
realtime weather view using Socket.io



### Clone and Run Locally
```
$ git clone https://github.com/KBPsystem777/realtime-weather

$ cd realtime-weather

$ npm install

$ npm run start
```
App will be available on http://localhost:3000

### Dependencies
```
"axios": "^0.18.0",
"socket.io": "^2.1.1"
```
